# VectCut 模板套版 安装指南

## Windows 安装

### 安装步骤

1. 从发布页下载 Windows 安装包，优先选择 `VectCut 模板套版-版本号-x64.exe`。
2. 双击安装包，按安装向导选择安装目录。
3. 安装完成后，通过桌面快捷方式或开始菜单启动 VectCut 模板套版。

### SmartScreen 拦截说明

MVP 内测阶段未购买代码签名证书，因此 Windows Defender SmartScreen 可能提示“Windows 已保护你的电脑”。这是未签名应用的常见提示，不代表安装包一定存在风险。

如果确认安装包来自项目发布页或可信内测渠道，可以按以下步骤继续：

1. 在 SmartScreen 提示窗口点击“更多信息”。
2. 确认应用名称和发布来源与下载渠道一致。
3. 点击“仍要运行”启动安装程序。

安全建议：VectCut 模板套版是面向内测的开源客户端，请只从官方仓库发布页或团队指定渠道下载安装包。不要运行来源不明、被二次打包或哈希不一致的文件。

### 系统要求

- Windows 10 64 位及以上。
- 剪映专业版 10.0-10.9。
- ffprobe 已内置，无需单独安装。

## macOS 安装

1. 从发布页下载 macOS `dmg` 文件。
2. 双击打开 `dmg`。
3. 将 `VectCut 模板套版` 拖到 `Applications` 文件夹。
4. 从 `Applications` 启动应用。

### 系统要求

- macOS 11 及以上。
- 剪映专业版 10.0-10.9（Mac 版）。
- ffprobe 已内置，无需单独安装。

### Gatekeeper 拦截说明

MVP 阶段 macOS 版本未签名、未公证。首次打开时，Gatekeeper 可能阻止启动。请在“系统设置”中进入“隐私与安全性”，找到被拦截的 VectCut 模板套版，并选择允许打开。

如果系统仍然拦截来自可信内测渠道的应用，可以打开终端运行以下命令移除隔离属性：

```bash
xattr -cr '/Applications/VectCut 模板套版.app'
```

只对来自官方仓库发布页或团队指定渠道的安装包执行此操作。不要运行来源不明、被二次打包或哈希不一致的文件。

## 首次使用

1. 启动 VectCut 模板套版后打开设置页。
2. 检查服务器地址，默认地址为 `https://api.vectcut.com`。如果团队提供了内测服务地址，请替换为对应地址。
3. 确认草稿目录自动检测结果。如果检测结果为空或不正确，手动选择剪映专业版草稿保存目录。
4. 点击测试连接，确认客户端可以访问服务端。
5. 点击保存设置。
6. 返回主界面，开始使用模板导入、素材配置和草稿生成功能。

## 服务端配置

首次启动后，打开设置页并配置以下内容：

- 服务器地址：默认地址为 `https://api.vectcut.com`，也可以填写本地 VectCutAPI 服务端地址，例如 `http://127.0.0.1:9001`，或团队提供的内测地址。
- 服务器认证：生产服务如果启用了 Basic Auth，填写团队提供的用户名和密码；本地开发服务通常可以留空。
- 剪映草稿目录：优先确认草稿目录自动检测结果；如果检测失败或路径不正确，手动选择本机剪映专业版草稿保存目录。

配置完成后，点击测试连接。测试通过后点击保存设置，再导入模板、配置素材并生成草稿。

## 常见问题

### SmartScreen 提示是否代表安装包有病毒？

不一定。MVP 内测阶段未购买代码签名证书，未签名安装包容易触发 SmartScreen。请核对下载来源，只运行官方仓库发布页或可信内测渠道提供的安装包。

### 找不到剪映草稿目录怎么办？

先打开剪映专业版并创建或保存一个草稿，再到剪映设置中查看草稿保存位置。回到 VectCut 模板套版设置页，选择该目录作为剪映草稿目录。

### 连接失败怎么办？

检查服务器地址是否填写完整，确认服务端已启动，并确认本机网络、防火墙或代理没有阻断访问。修改后在设置页重新点击测试连接。

### 生成后看不到草稿怎么办？

确认设置页中的剪映草稿目录与剪映专业版当前使用的草稿目录一致。生成完成后重启剪映专业版或刷新草稿列表，再检查是否出现新草稿。
