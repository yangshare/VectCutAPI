## 降级方案（如关键假设失败）

### 如果验证 1 失败（剪映无法打开草稿）

**可能原因**：
- pyJianYingDraft 版本过旧，不支持剪映 10.x 格式
- 草稿 JSON 缺少必需字段
- 草稿格式校验失败

**降级方案**：
1. 尝试升级 pyJianYingDraft 到最新版本
2. 对比手动修改的草稿与原始草稿的差异
3. 如果无法解决，考虑技术路线调整：
   - 改为"云端处理素材 + 返回完整草稿包"（成本高，但避开路径问题）
   - 或放弃元数据模式，改为"浏览器插件 + 本地脚本"

### 如果验证 2 失败（字幕样式无法继承）

**降级方案**：
- MVP 1.0 使用默认字幕样式（黑体/白色/底部居中）
- 让用户在剪映中手动调整字幕样式
- 后续迭代再优化样式继承
