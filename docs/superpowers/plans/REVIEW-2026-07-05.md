### ⚠️ 问题列表

#### Important

1. **本地文件读取安全限制**（任务 8.2 client.ts 第 1123 行）  
   ```typescript
   const fileBuffer = await (await fetch(`file://${zipPath}`)).blob();
   ```
   - **问题**：渲染进程通过 `file://` 协议读取本地文件可能受 CSP 或沙箱限制
   - **建议**：改为在主进程读取文件并通过 IPC 返回 Buffer

2. **ffprobe 打包路径未验证**（任务 5.1 + 任务 11.1）  
   - **问题**：`ffprobe-static` 的二进制路径在开发和打包后不同
   - electron-builder.yml 中 `extraResources` 的 `from` 路径需要验证是否正确
   - **建议**：在 mediaProbe.ts 中增加路径检测逻辑

3. **网络错误无重试机制**（任务 8）  
   - **问题**：API client 使用 axios 但未配置重试，长时间网络操作（上传大文件）可能因临时网络抖动失败
   - **建议**：配置 `axios-retry` 库

#### Minor

1. **硬编码路径分隔符**（任务 3.1）：`join(home, 'AppData/Local/...')` 混用了 `/`，虽然 Node.js 会自动处理，但不规范
2. **类型声明路径问题**（任务 2.3）：`global.d.ts` 导入 `'../electron/preload'` 时需要确认 tsconfig 的 `paths` 配置支持
3. **UI 组件无测试**（任务 9）：React 组件无单测，建议至少为核心交互逻辑增加 Vitest + React Testing Library 测试
4. **Windows PowerShell 命令转义**（任务 4.1）：`Compress-Archive -Path '${folderPath}/*'` 在路径含特殊字符时可能失败

### 🔧 改进建议

#### 修复 Important 问题（预计 1-2 小时）
